# 🎓 Student Performance Analyzer Using NumPy

A professional beginner-level **Python & NumPy** project that analyzes student academic performance using NumPy arrays. This project demonstrates the use of NumPy for data analysis, statistics, and student result processing.

---

## 📌 Features

- 📊 Store student marks using NumPy arrays
- 📚 Display student marks
- 📐 Display array properties
  - Shape
  - Dimension
  - Size
  - Data Type
- ➕ Calculate total marks
- 📈 Calculate average marks
- 🏆 Find class topper
- 📉 Find lowest scorer
- 📖 Find highest marks in each subject
- 📖 Find lowest marks in each subject
- ✅ Pass/Fail status
- 📝 Grade generation
- 📋 Final class report
